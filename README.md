## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/java-zero-to-advanced.git
```

### Navigate to the Project

```bash
cd java-zero-to-advanced
```

### Compile and Run

```bash
javac Main.java
java Main
```
